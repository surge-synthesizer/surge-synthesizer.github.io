We released Surge XT 1.2.3 on May 5, 2023. This release contains our **last** binary distribution of the LV2 format - 
moving forward, users who want to use the LV2 plugin will need to build it from the source.

This is a hotfix release which takes care of the following issue:

## UI/UX

-   Fixed a crash when selecting "Continue from current level" option in Play Mode right-click context menu
